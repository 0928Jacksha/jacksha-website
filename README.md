# Personal Website Skeleton

Clean multi-page website foundation focused on structure and navigation.

## Project Structure
- Home page (root): `index.html`
- Subpages: `pages/`
  - `travel.html`
  - `photography.html`
  - `work.html`
  - `life.html`
  - `about.html`
  - `contact.html`
- Assets: `assets/`
  - `css/main.css`
  - `js/main.js`
  - `images/`

## Notes
- All pages share the same main navbar.
- Styling is intentionally minimal for this phase.
